# Changelog

## [0.1.0] - 2026-03-21
### Added
- 기초 프로젝트 구상 및 React(Vite) 셋업.
- `PRD.md`, `MEMORY.md`, `README.md`, `CHANGELOG.md` 생성 및 영문 명칭 `Ko Woon Care` 로 통일.
- Vanilla CSS 기반 글래스모피즘(Glassmorphism) UI 및 따뜻한 파스텔톤 컬러 팔레트 구축.
- 한국 공휴일 및 대체휴일(`korean-holidays`) 로직 처리가 완벽히 연동된 달력 컴포넌트 렌더링.
- 월 단위 캘린더 날짜 클릭 시 노출되는 스와이프(iOS 룰렛 스타일) 시간 지정 팝업 제작. 
- 시간 입력 모달: 기본값(16:50~18:50) 세팅 기능, '돌봄 미제공' 처리 기능, 기존 일자 기록 삭제 및 덮어쓰기 기능 고도화.
- 달력 및 팝업 화면에 호랑이띠(`🐯`)를 상징하는 이모티콘 등 꾸미기 텍스트 반영.
- 월별 누적 근무 시간 및 급여(15,000원 기준) 자동 계산 대시보드 컴포넌트 작성. 
- 모바일 디바이스 지원에 최적화된 미디어 쿼리 작성 (수직 레이아웃 전환 및 UI 요소 리사이징).
- Supabase CRUD 로직 적용을 위한 `db_schema.sql` 스키마 초기화 스크립트 작성.
