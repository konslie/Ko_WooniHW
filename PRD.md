# PRD: 고운 하원 돌봄 관리 (Ko Woon Care)

## 1. 개요 (What & Why)
- **목적**: 운이의 하원 및 돌봄을 담당하시는 선생님의 근무 시간을 기록하고 월별 급여 정산을 자동화하여 투명하고 편리한 관리를 도모.
- **주요 대상**: 운이 부모님 (관리자용)

## 2. 핵심 기능
1. **캘린더 기반 근무시간 이력 관리**
   - 월 단위 캘린더 제공.
   - 한국 공휴일 정보 연동 및 주말/공휴일 시각적 차별화.
   - 캘린더 칸 안에 입력된 근무 시간 정보 직관적 노출.
2. **근무시간 입력 기능 및 특이사항 메모 (Time Picker & Memos)**
   - 캘린더 날짜 클릭 시 시간 입력 팝업 렌더링.
   - 하원 돌봄(제공시간/미제공) 및 관련된 등원 특이사항(예: 체험학습) 메모를 상단 3-탭 구조 안에서 독립적이면서도 자유롭게 혼용 기재 가능.
   - `00:00 - 00:00` (From - To) 형식. 상하 스와이프 혹은 스크롤을 통한 시/분 조작 (iOS 휠 스타일).
   - OCR 기능 (추후 반영): 프린트된 캘린더 사진 내 시간 텍스트(예: "4:55~5:55") 자동 추출. 하원 시간 특성을 반영하여 모든 시간은 기본적으로 **오후(PM)**로 자동 간주함.
3. **월별 정산 현황판 (Dashboard)**
   - 페이지 우측에 상시 위치.
   - 해당 월의 총 돌봄 시간 및 총 정산 금액(기본 15,000원/시간 연동) 통계 제공.
   - 화면 레이아웃: 캘린더(주요 기능 영역) 75%, 현황판 25% 비율.

## 3. 비기능적 요구사항 (Non-Functional)
- **Tech Stack**: Vercel (Hosting), Supabase (DB & Auth), Frontend (Vite + React 추천).
- **Design**: 아이 하원 돌봄의 따뜻하고 귀여운 느낌을 살린 전용 컬러 테마(예: 파스텔 톤 옐로우/오렌지/그린 등) 적용. 최신 웹 트렌드(글래스모피즘, 부드러운 애니메이션)를 준수한 프리미엄 UI, Vanilla CSS 사용.
