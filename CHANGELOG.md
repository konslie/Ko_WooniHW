# Changelog

## [0.3.0] - 2026-04-03
### Added
- 돌봄 달력 내 하위 범주인 '특이사항' 항목 추가 (체험학습 및 특이사항 등 등원 관련 메모 기능).
- 캘린더 칸 하단에 '특이사항' 및 상세 메모를 돌봄 미제공과 동일한 통일된 포맷의 전용 UI(녹색 뱃지)로 표시하도록 추가.
- `care_logs` Supabase DB 테이블 스키마 수정 없이 기존 `memo` 필드를 활용하는 파싱 로직 적용.

### Fixed
- 특이사항 항목 입력 시 기존 입력된 돌봄 제공/미제공 내역을 덮어쓰는 버그 원천 수정 (두 기능이 완전히 독립적으로 동작하고 하이브리드 JSON 형태로 `memo` 필드에 동시 저장 및 파싱되도록 개선).

## [0.2.0] - 2026-03-21
### Added
- Vercel 자동 배포 연동 및 Supabase 환경변수 반영.
- 카카오톡 이미지 공유 기능 추가 (상단 🐯 헤더의 공유 버튼 클릭 시 캘린더 화면 통째로 렌더링 후 공유).

### Changed
- 모바일 환경에서의 캘린더 헤더 UI 최적화 (`flex-direction: column` 래핑 적용으로 화살표 버튼 짤림 버그 수정).
- 달력 내 배지(돌봄 시간, 돌봄 없는 날)의 내부 패딩을 0으로 없애고 좌우 둥근 모서리(border-radius)를 3px로 축소하여 텍스트 표시 가능 영역 극대화.
- 컨테이너 쿼리(cqw) 허용 범위를 파격적으로 넓혀 모바일 화면에서도 텍스트가 달력 칸을 가득 채우게 개선.
- 모바일 세로 뷰포트에서 달력과 하단 대시보드가 스크롤 없이 **딱 한 화면**에 들어오도록 불필요한 마진(margin) 및 패딩 대폭 압축.
- 공유 기능 품질 향상을 위해 `html2canvas` 캡쳐 배율을 2x에서 4x로 대폭 업그레이드.
- 브라우저 탭 타이틀을 `temp-app`에서 `🐯 운이 돌봄달력`으로 변경하고 HTML 언어 설정을 `ko`로 지정.

### Removed
- 대시보드의 미사용 하드코딩 버튼("💌 카카오톡 공유") 삭제 후 헤더 아이콘 버튼 하나로 기능 일원화.

## [0.1.0] - 2026-03-21
### Added
- 기초 프로젝트 구상 및 React(Vite) 셋업.
- `PRD.md`, `MEMORY.md`, `README.md`, `CHANGELOG.md` 생성 및 영문 명칭 `Ko Woon Care` 로 통일.
- Vanilla CSS 기반 글래스모피즘(Glassmorphism) UI 및 따뜻한 파스텔톤 컬러 팔레트 구축.
- 한국 공휴일 및 대체휴일(`korean-holidays`) 로직 처리가 완벽히 연동된 달력 컴포넌트 렌더링.
- 월 단위 캘린더 날짜 클릭 시 노출되는 스와이프(iOS 룰렛 스타일) 시간 지정 팝업 제작. 
- 시간 입력 모달: 기본값(16:50~18:50) 세팅 기능, '돌봄 미제공' 처리 기능, 기존 일자 기록 삭제 및 덮어쓰기 기능 고도화.
- 달력 및 팝업 화면에 호랑이띠(`🐯`)를 상징하는 이모티콘 등 꾸미기 텍스트 반영.
- 월별 누적 근무 시간 및 급여(15,000원 기준) 자동 계산 대시보드 컴포넌트 작성. 
- 모바일 디바이스 지원에 최적화된 미디어 쿼리 작성 (수직 레이아웃 전환 및 UI 요소 리사이징).
- Supabase CRUD 로직 적용을 위한 `db_schema.sql` 스키마 초기화 스크립트 작성.
