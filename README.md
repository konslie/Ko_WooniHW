# 고운 하원 돌봄 관리 (Ko Woon Care)

운이의 하원 및 방과 후 돌봄을 담당하시는 선생님의 일별 스케줄을 관리하고, 근무 시간에 따른 월급 정산을 빠르고 편리하게 자동화하는 웹 애플리케이션입니다.

## 주요 기능
- **캘린더 관리**: 한국 공휴일 및 대체공휴일(`korean-holidays` 적용)이 자동 반영된 달력을 통해 직관적으로 근무 일자 파악.
- **스와이프 시간 입력**: 날짜 클릭 시, 상하 스와이프 방식의 iOS 룰렛 뷰로 간편하고 빠르게 시간 지정 (기본값 4:50 - 6:50 PM). 
- **돌봄 미제공 및 삭제**: 돌봄이 없는 날은 사유를 기록하여 캘린더에 별도 배지로 렌더링. 이미 입력된 기록도 간편하게 팝업 내에서 바로 삭제 가능.
- **대시보드 리포팅**: 월별 총 돌봄 시간 및 현행 시급에 따른 예상 정산 금액 자동 합산.
- **모바일 대응 프리미엄 UI**: 작은 모바일 화면에서도 깨지지 않으며 투명한 글래스 질감과 아기자기한 이모티콘 테마 적용.

## 기술 스택
- **Frontend**: Vite (React), Vanilla CSS, date-fns, lucide-react
- **Backend / Database**: Supabase 
- **Hosting**: Vercel

## 실행 방법
```bash
npm install
npm run dev
```
